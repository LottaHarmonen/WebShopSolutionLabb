# WebShopSolutionLabb

## Overview
WebShopSolutionLabb is a backend solution for a webshop, designed as part of a school project to demonstrate clean architecture, test-driven development (TDD), 
and the application of key design patterns such as Abstract Factory, Observer, and the Generic Repository pattern. The solution includes in-memory 
database support and API documentation powered by Swagger, allowing users to explore and test all API endpoints interactively.
